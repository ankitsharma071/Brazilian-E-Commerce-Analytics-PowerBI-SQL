# 📊 Brazilian E-Commerce Analytics Dashboard

An end-to-end Business Intelligence project built using **MySQL** and **Power BI** on the Brazilian E-Commerce (Olist) dataset. This project focuses on transforming raw transactional data into actionable business insights through advanced SQL analysis and interactive dashboards.

---

## 🚀 Project Overview

This project analyzes customer behavior, sales performance, and marketplace operations using over 100,000 e-commerce orders from the Olist dataset.

The analysis includes:

- Executive Sales Dashboard
- Customer Analytics Dashboard
- RFM (Recency, Frequency, Monetary) Segmentation
- Advanced SQL Analysis
- Business Insights & KPI Reporting

# 📊 Key Business Insights

- Top-selling states contribute the majority of marketplace revenue.
- Repeat Customer Rate is only **3.12%**, highlighting customer retention opportunities.
- At-Risk customers generate significant revenue and should be prioritized for retention campaigns.
- RFM Segmentation helps identify Champions, Loyal, New, Regular, and At-Risk customers.
- Logistics-heavy product categories incur significantly higher freight costs.
- Top 544 Sellers Generate 80% of the Total Revenue, following the **Pareto (80/20) Principle**.
